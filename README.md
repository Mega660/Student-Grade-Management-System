#  Student Grade Management System

A console-based Java application for managing student records, calculating averages, and generating letter grades automatically.

Users can:

* Add students
* Assign grades (0–100)
* Calculate average scores
* Automatically determine letter grades based on a grading scale

This project demonstrates core Java programming concepts such as Object-Oriented Programming (OOP), ArrayLists, input validation, and structured menu-driven application design.

## Features

*  Add new students
*  Enter grades (0–100)
*  Automatic average calculation
*  Automatic letter grade generation (A–F)
*  Display all student records
*  Grade validation system
*  Continuous interactive menu

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Switch Expressions (Java 14+)

## How to Run

1. Install Java (JDK 14 or higher recommended)

2. Compile the program:

javac StudentGradeSystem.java

3. Run the program:

java StudentGradeSystem
